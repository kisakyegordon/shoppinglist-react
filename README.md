ShoppingList React App
========

## About
This is a React Application, created using Facebook's create-react-app.
The App uses a flask/python API.

## Instructions
- Clone this repository
- Run npm install
- npm run start

## Run Tests
- npm test
 With Coverage
 - npm test -- --coverage
